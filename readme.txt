HTMLで文字詰め（カーニング）するプリロード
バージョン: 0.01 掲載日: 2013/9/15

====================
説明
====================
その名のとおり、HTMLで文字詰め（カーニング）するプリロードです。
下記で公開されているライブラリを利用させていただきました。（感謝）

解凍して出来上がったhtmlディレクトリにあるcommon/FLAutoKerning ディレクトリ以下のファイル群とpreload/FLAutoKerning.class.php ファイルをサイトにアップしてください。

h1〜h4とpタグについて、自動でカーニングを行うようにしてみました。
もし、カーニングが必要ない場合は、各タグに class="no" を追加してください。
また、カーニングさせたい文字を変更・追加したい場合は、直接　FLAutoKerning.js を編集してください。

オリジナルJS配布サイト（ライセンスCC）
　http://fladdict.net/blog/2011/02/auto-kerning.html
上記のカスタマイズ版配布サイト（ライセンスCC）
　https://gist.github.com/nijitaro/846898
